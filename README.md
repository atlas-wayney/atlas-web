# ATLAS Atlas Web

A demonstration web application for Atlas showcasing user experiences and workflows for managing access controls and network configurations.

## Features

### Access Matrix
- **Manage Client Groups** - Create and manage client group configurations
- **Manage Clients** - Add, edit, and manage client information
- **Manage Users** - Manage user accounts and permissions
- **Manage Access Matrix** - Configure access controls and permissions matrix

### Network
- **Colo Onboarding** - Manage colocation facility onboarding and setup
- **ST Market Access** - Securities trading market access configuration
- **DT Market Access** - Derivatives trading market access configuration
- **DC Market Access** - Derivatives clearing market access configuration

### Billing
- **Billing Scheme** - Configure and manage billing scheme definitions
- **Billing Calculation** - Calculate and review billing amounts
- **Billing Report** - Generate and view billing reports

## Tech Stack

- Next.js 16 with Turbopack
- React 19
- TypeScript
- Tailwind CSS 4
- HeroUI (component library)
- Nivo (charts)
- React Flow (flow diagrams)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Scripts

- `npm run dev` - Start development server with Turbopack
- `npm run build` - Build for production (static export)
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
